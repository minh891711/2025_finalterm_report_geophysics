HW6：地震層析成像演講心得

講者：甘禮有 博士 日期：2025/10/01

地震波與成像基礎

演講由淺入深，從層析成像開始介紹地震波的傳遞路徑。博士詳細拆解了地震研究的三大要素： Source (震源)、Structure (構造) 與 Receiver (接收器)。

特別引人入勝的是關於 Forward Problem (正演) 與 Inverse Problem (反演) 的探討，這與我們第一週學到的模擬概念相呼應，讓我對地震物理的計算邏輯有了更具體的認識。

臺灣地底的波速視界

透過波速圖觀察到的地質特徵：西部地區為低速帶，中央山脈則呈現高速帶（變質岩特徵），並能清晰辨識大屯火山與龜山島的岩漿庫。

演講展示了與傳統課本不同的板塊觀點：菲律賓海板塊隱沒至歐亞板塊下方時，中間存在顯著的空間，且西南部的交界處以淺藍色標示出海洋板塊。這些視覺化證據讓我對臺灣的地質構造有了全新的理解。

總結與啟發

雖然地震學中許多深奧的物理量對現在的我來說仍具挑戰性，但這次演講透過大量的圖片視覺化，讓枯燥的數據變成生動的地層剖面。 這讓我更進一步體會到地球物理不僅是公式，更是探索腳下未知世界的眼睛。
