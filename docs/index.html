校園折射探勘實驗報告 (HW5)

實驗原理：司乃耳定律與走時

司乃耳定律（Snell's Law）是描述波在不同介質界面折射現象的核心公式。當地震波穿過地表，遇到波速不同的地層界面時，其入射角與折射角的正弦值之比等於波速之比。

在折射探勘中，我們利用地層波速隨深度增加的特性，觀察波速較快的下層界面產生的「臨界折射波」。透過分析地表受波器（Geophone）接收到波的時間（走時，Travel Time）與距離的關係，我們可以推算出各層的深度、厚度及真實波速。

所需器材

  • 受波器 (Geophones)
  • 地震儀傳輸纜線
  • 打擊震源（落槌或大鎚）
  • 高精度 GPS 定位儀器
  • 數位訊號記錄主機

實驗心得

這次在校園中進行的折射探勘實驗是一次非常寶貴的實務經驗。將課本上抽象的「司乃耳定律」轉化為實際佈設受波器、敲擊落槌，並看著訊號在地震儀主機上即時產生的過程，讓我對地震波在地下介質中的傳播有了全新的認識。在野外操作中，我們學會了如何排除校園背景噪訊的干擾，以及與團隊成員協作確保每一條纜線的精確對接。雖然過程辛苦，但最後分析走時曲線得出地下層位的瞬間，那種成就感是難以言喻的。這不僅是一場科學實驗，更是一次結合耐心、體力與腦力的地球物理實踐。
