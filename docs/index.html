給同學的一封信

各位同學，歡迎來到這趟充滿驚喜的地球物理之旅。

你是否曾好奇過，在我們腳下幾十公里、甚至幾千公里的地方，究竟發生了什麼事？我們無法直接挖掘到地核，但我們有物理與電腦。

這門課不僅會教你傳統的探勘理論，更會帶領你進入 AI 與自動化觀測 的殿堂。我們會使用 Python、Gemini API 與 Dify，親手打造屬於你的地球監測應用程式。讓我們一起從數據中，看穿地球的秘密。
