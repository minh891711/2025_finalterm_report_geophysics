課程介紹

本課程旨在引導學生探索地球內部的運作規律，並掌握現代地球物理學家必備的數位武器。我們將學習內容分為兩大核心模組：

地球物理知識

涵蓋折射探勘、重力探勘、地震學、板塊構造、地磁與地熱。從理論基礎到實際應用案例分析。

資料處理技能

手把手教導 Python 環境、Gemini API、Dify、Gradio、Hugging Face 等尖端 AI 工具，以及 PyGMT、ObsPy 等專業地科軟體。
