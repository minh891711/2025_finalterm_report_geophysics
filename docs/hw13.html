HW13：旅日經驗與地球物理實際應用

講者：黃有志 博士 日期：2025/12/10

地震層析與觀測基礎

這次演講內容非常豐富，特別是結合了大量的圖片，讓複雜的科學內容變得易於理解且不枯燥。 黃博士從層析成像的基本原理談起，詳細介紹了地震波在地球內部的傳遞機制，並清晰地拆解了 Source (震源)、Structure (構造) 與 Receiver (接收器) 三者間的關係。

最令我印象深刻的是 Forward Problem (正演) 與 Inverse Problem (反演) 的部分， 因為這與我們第一週課程提到的模擬概念產生了強烈連結，讓我更能掌握實際應用中的計算邏輯。

臺灣與日本的地底視界

演講末尾展示了臺灣地底深度的波速圖，可以清楚看到 西部低速帶 與 中央山脈高速帶（由變質岩組成）的對比，甚至還能辨識出大屯火山與龜山島的岩漿庫構造。

特別驚訝於一張關於板塊構造的細節圖：菲律賓海板塊隱沒至歐亞板塊下方時，中間展現出巨大的空間， 而西南交界處的淺藍色部分則標示著海洋板塊的特徵。這與過去課本上的簡化圖示有很大的不同。

心得總結

雖然目前還有許多專業細節無法完全看透，但透過這次演講，我對阿蘇火山到嘉南地區的地震觀測實務有了全新的認識。 這場演講讓我對地球物理知識的實際應用產生了更大的興趣，也期待未來能更深入地探索這個領域。
