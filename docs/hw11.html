PyGMT 快速入門

PyGMT 是什麼？為什麼你會愛上它？

簡單說：PyGMT＝GMT 全部功能＋純 Python 語法
你原本要花好幾十行 GMT 指令才能畫出的論文級地圖，現在用 Python 幾行就搞定！

完全支援 NumPy、Pandas、Xarray
直接在 Jupyter Notebook 裡「改參數 → 立刻看結果」
內建全球高解析度地形、海齡、衛星圖……完全不用自己下載！

哪些人最適合學 PyGMT？

  1. 手邊有大量的地理數據需要呈現在地圖上
  2. 時常繪製地圖，並且總是覺得使用 GIS 軟體效率太低
  3. 需要即時或互動性呈現地理資料分析的結果

用 Python 的語法，畫出比商業軟體還專業的地圖 —— 這就是 PyGMT

全球地形熱圖（Robinson 投影）

三行程式碼，無需下載資料，即可繪製專業級全球地形

fig = pygmt.Figure() fig.grdimage(grid=grid, projection="R12c") fig.show()

海底地形 + 等高線 + 色階條

haxby 配色 + grdcontour，輕鬆做出論文級海底地形圖

fig = pygmt.Figure() fig.grdimage(grid=grid, cmap="haxby", projection="M10c", frame=True) fig.grdcontour(grid=grid, annotation=1000, levels=250, limit=[-4000, -2000]) fig.colorbar(frame=["x+lelevation", "y+lm"]) fig.show()

3D 立體地形視圖

grdview + perspective 一鍵生成電影級 3D 地形

fig = pygmt.Figure() fig.grdview( grid=grid, perspective=[-130, 30], frame=["xaf", "yaf", "WSnE"], projection="M15c", zsize="1.5c", surftype="s", cmap="geo", plane="1000+ggrey", contourpen="0.1p", ) fig.colorbar(perspective=True, frame=["a500", "x+lElevation", "y+lm"]) fig.show()

海岸線地圖 + 嵌入小地圖（Inset）

fig.coast 結合 fig.inset，輕鬆做出帶定位框的區域地圖

fig = pygmt.Figure() fig.coast( region=[-74, -69.5, 41, 43], borders="2/thin", shorelines="thin", projection="M15c", land="lightyellow", water="lightblue", frame="a", ) with fig.inset( position="jBL+o0.5c/0.2c", box=Box(pen="black"), region=[-80, -65, 35, 50], projection="M3c", ): fig.coast( land="gray", borders=[1, 2], shorelines="1/thin", water="white", dcw="US.MA+gred", ) fig.show()
